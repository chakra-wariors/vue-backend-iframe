# 使用

## 依赖

**node** > 10
**yarn** 安装nodejs完成后，命令行执行 `npm install yarn -g`

## 目录执行`yarn`

## 本地启动 `yarn dev`

## 本地打包构建 `yarn build`

## 配置
`src/router/routers.js`文件里对左侧、顶部菜单配置

`export navs`表示顶部菜单

`export default`表示左侧菜单
